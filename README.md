# CourierBot



## О проекте

**CourierBot**  —  Информационный помощник для курьеров с привычным интерфейсом.

Помогает подготовить необходимую для работы информацию.

Например, пропишет адреса в список отгрузок — это сэкономит время и силы при планировании порядка доставки и загрузки.



## Локальный запуск

- Клонируйте репозиторий:
```shell
git clone <SSH ссылка на репозиторий>
```

- Создайте файл `.env`. Для этого, находясь в корне проекта, выполните:
```shell
cp .env.exemple .env
```

- Чтобы создать своего Telegram бота, найдите официального бота @BotFather и следуйте его инструкциям, или просто последовательно отправьте ему сообщения:
   - `/start`
   - `/newbot`
   - `<Название вашего бота>`
   - `<Уникальный username вашего бота>`

- Скопируйте токен, который выдал вам @BotFather

- Откройте `.env` и вставьте токен в переменную `TOKEN`

- Запустите программу:
```shell
python main.py
```

- Готово, бот активен!

- Для завершения его работы, находясь в терминале, нажмите `Ctrl+C`
