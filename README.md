# CourierBot



## О проекте

**CourierBot**  —  Информационный помощник для курьеров с привычным интерфейсом.

Помогает подготовить необходимую для работы информацию.

Например, сам выпишет отгрузки для конкретного курьера из общего списка и пропишет к ним адреса — это сэкономит время при планировании порядка доставки и загрузки.
